## Hook - useWalletClient

- required provider: [**ChainProvider**](https://docs.cosmoskit.com/chain-provider)

- parameters:
    - **walletName?**: `WalletName` ( = `string` );

> if `walletName` is undefined, using `currentWallet` instead. See LocalStorage key `cosmos-kit@1:core//current-wallet`

- return type: [**WalletClientContext**](#type---walletclientcontext)

## Type - WalletClientContext

### Properties

| Name      | Description | Type | Default |

| **client**   | wallet client     | `[WalletClient](https://github.com/cosmology-tech/cosmos-kit/blob/edc8e7e176d388ce2cb3f94a6d05d6c1949c3662/packages/core/src/types/wallet.ts#L114-L164) \| undefined` | `undefined` |
| **status**   | wallet client status     | `State` | `State.Init` |
| **message**   | error message if wallet client is undefined    | `string \| undefined` | `undefined` |

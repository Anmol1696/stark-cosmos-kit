# How to Add Fin Wallet to CosmosKit

There are two packages for fin

- `@cosmoskit/fin`
- `@cosmoskit/fin-extension`

`@cosmoskit/fin` export all available fin wallets (currently only extension available), while if you only want to add a particular one, choose `@cosmoskit/fin-extension`

> Note: all these packages export `wallets` and it's an array of `MainWalletBase`

Take `@cosmoskit/fin` for example

## add `@cosmoskit/fin`

```
yarn add @cosmoskit/fin
```

## import the wallets

```js
import { wallets as fin } from "@cosmos-kit/fin";
```

## add to your provider

```js
function MyCosmosApp({ Component, pageProps }: AppProps) {
  return (
      <ChainProvider
        chains={chains}
        assetLists={assets}
        wallets={[...fin]}
        walletConnectOptions={...} // required if `wallets` contains mobile wallets
      >
        <Component {...pageProps} />
      </ChainProvider>
  );
}

export default MyCosmosApp;
```
